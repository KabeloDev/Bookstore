 ### [YouTube Demonstration part 1](https://youtu.be/jutWwzjbdqU)
 ### [YouTube Demonstration part 2](https://youtu.be/6oUkfj-FKh8)

<h2>Description</h2>
A book store demo app where users can view books, add them to a cart, buy and track their orders
<br />


<h2>Languages and Utilities Used</h2>

- <b>ASP.NET Core Web API</b>
- <b>ASP.NET Core MVC</b>
- <b>MS SQL Server</b>
- <b>Bootstrap</b>

<h2>Environments Used </h2>

- <b>Windows 10</b>
- <b>Visual studio 2022</b>
